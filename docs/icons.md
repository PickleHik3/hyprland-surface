# Icons

Dank Material Shell icon names are Material Symbols names.

Use this browser when choosing icons:

https://fonts.google.com/icons

Current `surface-dms` icons:

- Back: `arrow_back_ios`
- Recent Apps: `square`
- Keyboard auto hidden: `keyboard`
- Keyboard visible: `keyboard_hide`
- Keyboard disabled: `keyboard_off`

These are not Nerd Font glyph names.
